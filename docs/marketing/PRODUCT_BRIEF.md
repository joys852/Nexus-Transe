# NexusIDE — 产品介绍

## 一句话

NexusIDE 是面向专业开发者的 **AI 原生开发体验**：以 **CLI + 本地引擎** 为核心，多模型、可扩展、企业级安全。

## 核心卖点

1. **终端优先** — 交互式 REPL 与工具循环
2. **真·工具执行** — 文件/Git/Shell 在本地沙箱执行，可审批
3. **生产就绪** — 加密密钥、审计日志、CI/CD、跨平台 CLI 构建
4. **开放生态** — MCP + 插件 + 多 LLM 提供商

## 目标用户

- 全栈工程师、Tech Lead
- 需要本地优先与合规的企业团队
- 需要可自托管、本地优先的终端开发助手团队

## 截图脚本（营销素材）

1. 深色主题终端：`nexus chat` 流式输出
2. 斜杠命令与工作流（如 `/plan`、`/sessions`）
3. 工具审批提示与说明
4. `nexus index` 索引进度
5. `nexus provider list` 多模型配置
