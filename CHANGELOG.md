# Changelog

All notable changes to [Nexus-Transe](https://github.com/joys852/Nexus-Transe) are documented here.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.1.0/).

## [1.0.0] - 2025-05-17

### Added

- Production CLI distribution (`nexus` / `nx`) with auto engine bootstrap
- Python LangGraph engine sidecar (HTTP/SSE, MCP, vector search)
- Install scripts: `scripts/install.ps1`, `scripts/package-release.ps1`
- Apache-2.0 license, NOTICE, branding logo (`assets/logo.png`)
- Documentation: INSTALL, DISTRIBUTION, PRODUCTION, RELEASE

### Changed

- Project branding: **Nexus-Transe** (Cybertron Nexus Command Interface)
- Removed Web/Desktop shells; CLI-only focus
